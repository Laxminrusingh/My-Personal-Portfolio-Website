# Laxmi Nrusingh Rath - Portfolio Website

A modern, responsive portfolio website built with React and Vite, showcasing my skills, projects, and experience as a Full Stack Developer and Machine Learning enthusiast.

## Features

- **Responsive Design** - Fully responsive across all devices
- **Modern UI/UX** - Clean and attractive interface with smooth animations
- **Interactive Sections**:
  - Home/Hero section with profile and introduction
  - About Me section highlighting key traits
  - Education timeline
  - Technical Skills categorized by domain
  - Professional Experience & Internships
  - Project Showcase with detailed descriptions
  - Contact Form with real-time status feedback
- **Resume Download** - Direct download functionality
- **Smooth Animations** - Powered by Framer Motion
- **Modern Navbar** - Responsive navigation with glassmorphism effects
- **Contact Form** - Integrated with Formspree for message handling

## Tech Stack

- **Frontend Framework**: React 19.2.0
- **Build Tool**: Vite 7.2.4
- **Routing**: React Router DOM 7.10.1
- **Animations**: Framer Motion 12.23.25
- **Icons**: React Icons 5.5.0
- **Styling**: CSS-in-JS (Inline Styles)
- **Form Handling**: Formspree

## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Portofolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
